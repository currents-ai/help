# Connect Your WordPress Site via Plugin

This guide provides a step-by-step walkthrough for connecting your WordPress website to the platform using the dedicated ACME.BOT WordPress plugin. This method streamlines the authentication process and allows seamless content publishing directly to your site.

### 1. Navigate to Connectors

From the main dashboard, expand the **Resources** tab in the left-hand navigation menu. Click on the **Connectors** link to proceed.

![Navigate to Connectors](/img/help/connectors/wordpress-plugin/step-1.png)

### 2. Create a New Connector

On the Connectors page, click the **+ Create Connector** button located in the top-right corner to begin the setup process.

![Create a New Connector](/img/help/connectors/wordpress-plugin/step-2.png)

### 3. Select Connector Type

From the "Connect Your Blogging Platform" screen, click the dropdown menu and select **Connector Wordpress Plugin** from the list of available options. Click **Next** to continue.

![Select Connector Type](/img/help/connectors/wordpress-plugin/step-3.png)

### 4. Enter Your Website URL

Enter the full URL of your WordPress website into the "Website" field. Click the **Install Plugin in Wordpress** button, which will open the official plugin page in a new browser tab.

![Enter Your Website URL](/img/help/connectors/wordpress-plugin/step-4.png)

### 5. Install the Plugin

In the new tab displaying the WordPress plugin page, click the **Install Now** button to add the ACME.BOT plugin to your website.

![Install the Plugin](/img/help/connectors/wordpress-plugin/step-5.png)

### 6. Activate the Plugin

After the installation is complete, the button will change to **Activate**. Click it to enable the plugin on your WordPress site.

![Activate the Plugin](/img/help/connectors/wordpress-plugin/step-6.png)

### 7. Initiate Connection from WordPress

You will be redirected to the plugin's settings page within your WordPress admin area. Click the **CONNECT TO ACME BOT** button to authorize the connection.

![Initiate Connection from WordPress](/img/help/connectors/wordpress-plugin/step-7.png)

### 8. Name and Save the Connector

After authorizing, you will be redirected back to the ACME.BOT platform. Enter a descriptive name for your new connection in the "Connector Name" field and click **Save Connector**.

![Name and Save the Connector](/img/help/connectors/wordpress-plugin/step-8.png)

### 9. Confirm Connection

You will be taken back to the main Connectors page, where your newly created WordPress Plugin Connector will now be listed, confirming the connection is successful.

![Confirm Connection](/img/help/connectors/wordpress-plugin/step-9.png)
